# hubot-gglurl

hubot script for Google UrlShortener 

[![NPM version](https://badge.fury.io/js/hubot-gglurl.svg)](http://badge.fury.io/js/hubot-gglurl)

## Usage

Encode:

`hubot gglurl encode <url>`

Decode:

`hubot gglurl decode <url>`


## Install

`npm install hubot-gglurl`
